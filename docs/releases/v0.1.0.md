# v0.1.0

## 中文

PikPak Auto Bangumi 首个公开版本，聚焦于把番剧订阅、RSS 规则、PikPak 投递、重命名和网页管理统一到同一套 Bun 运行时中。

### 亮点

- 新增 Bangumi 收藏看板与设置中心网页界面
- 建立以 `bangumiSubjectId` 为核心的订阅绑定模型
- 支持 Mikan 辅助订阅与手动 RSS 订阅两条路径
- 打通 RSS 拉取、过滤、历史回放、PikPak 投递、云端重命名与弹幕补传
- 提供 CLI 模式与 Web UI 模式，兼顾自动化运行与本地管理
- 附带 Windows `start.bat` 启动脚本，便于本地一键拉起

### 适合谁

- 已经使用 Bangumi 管理追番状态的用户
- 想把 RSS 更新自动投递到 PikPak 的用户
- 希望保留可视化配置界面，同时又需要脚本化调度能力的用户

## English

The first public release of PikPak Auto Bangumi focuses on unifying anime subscription management, RSS filtering, PikPak delivery, cloud rename, and a Web UI inside one Bun-based runtime.

### Highlights

- Introduces a Bangumi collection board and a settings center in the Web UI
- Uses a subject-bound subscription model centered on `bangumiSubjectId`
- Supports both Mikan-assisted subscriptions and manual RSS subscriptions
- Connects RSS polling, filtering, replay, PikPak delivery, rename, and danmaku refresh in one pipeline
- Ships both CLI mode and Web UI mode for automation and local management
- Includes a Windows `start.bat` launcher for one-step local startup

### Best For

- Users who already track anime with Bangumi
- Users who want RSS updates delivered to PikPak automatically
- Users who want both a visual configuration panel and a scriptable runtime
